## Getting Started
- Copy this project into your local machine
- Install npm
```
npm install
```
- Create a .env file in the root of your project 
- Write key and value pairs
```
REACT_APP_NASA_KEY=<YOUR_API_KEY>
```
- Launch app
```
npm start
```
